# 三切面隔离检查工具实施计划

## 项目概述

开发一个桌面工具，用于检查API的三切面隔离状态。工具通过上传Excel文件，解析其中的API信息，发送HTTP请求并根据响应码判断隔离状态。

## 技术栈选择

* **编程语言**: Python 3.8+

* **GUI框架**: PyQt5 (成熟稳定，支持跨平台)

* **Excel处理**: pandas + openpyxl

* **HTTP请求**: requests

* **打包工具**: PyInstaller (生成.exe文件)

## 功能需求分析

### 1. 界面设计要求

* 大量留白设计，简洁现代

* 单色调配色方案（建议使用蓝/灰/白）

* 精简UI元素，避免视觉混乱

* 清晰的信息层级和排版

### 2. 核心功能模块

* **文件上传模块**: 支持拖拽上传和按钮上传Excel文件

* **Excel解析模块**: 解析必需字段并处理域名列的多域名情况

* **HTTP请求模块**: 拼接URI并发送请求

* **隔离判断模块**: 根据切面分类和响应码判断隔离状态

* **结果显示模块**: 以表格形式显示结果，使用颜色标识状态

### 3. 数据字段要求

必需字段：

* 切面分类（用户访问面/运行集成面/运维管理面）

* 域名（支持多域名，用分隔符分隔）

* API（API路径）

* 请求方式（GET/POST/PUT/DELETE等）

## 实施步骤

### 阶段一：项目初始化与环境搭建

1. 创建项目目录结构
2. 初始化Python虚拟环境
3. 安装依赖包（PyQt5, pandas, openpyxl, requests, PyInstaller）
4. 创建requirements.txt文件

### 阶段二：GUI界面设计与实现

1. 设计主窗口布局

   * 文件上传区域（拖拽区域 + 选择按钮）

   * 控制按钮区域（开始检查、停止、导出结果）

   * 结果显示区域（表格视图）

   * 状态显示区域（进度条和状态信息）

2. 实现UI组件

   * 主窗口类继承QMainWindow

   * 使用QTableWidget显示结果

   * 实现拖拽文件上传功能

   * 设计单色调配色方案

### 阶段三：Excel文件解析模块

1. 实现Excel文件读取

   * 使用pandas读取Excel文件

   * 验证必需字段是否存在

   * 处理缺失字段的异常情况

2. 域名解析功能

   * 识别域名列中的分隔符（逗号、分号、空格等）

   * 将多域名拆分为单个域名列表

   * 为每个域名生成独立的检查任务

3. URI拼接逻辑

   * 域名 + API路径 = 完整URI

   * 处理URL格式问题（添加http/https前缀）

   * 验证URI的有效性

### 阶段四：HTTP请求与隔离判断模块

1. HTTP请求发送

   * 根据请求方式发送对应类型的请求

   * 设置合理的超时时间

   * 处理网络异常和连接错误

2. 隔离判断逻辑

   * **运行集成面API**:

     * 响应码403 → "运行集成面API已隔离"（绿色加粗）

     * 其他响应码 → "运行集成面API未隔离"（红色加粗）

   * **运维管理面API**:

     * 响应码403 → "运维管理面API已隔离"（绿色加粗）

     * 其他响应码 → "运维管理面API未隔离"（红色加粗）

3. 结果记录

   * 记录每个API的检查结果

   * 保存响应码、响应时间、错误信息

   * 统计成功/失败数量

### 阶段五：结果显示与导出功能

1. 表格显示优化

   * 使用QTableWidget显示检查结果

   * 根据状态设置单元格颜色和字体

   * 实现表格排序和筛选功能

2. 结果导出

   * 支持导出为Excel文件

   * 保持原始格式并添加检查结果列

   * 导出统计报告

3. 进度反馈

   * 实时显示检查进度

   * 显示当前检查的API信息

   * 统计信息更新

### 阶段六：多线程与性能优化

1. 多线程实现

   * 使用QThread避免界面卡顿

   * 实现并发请求控制（限制同时请求数量）

   * 线程安全的数据访问

2. 性能优化

   * 请求超时设置

   * 连接复用

   * 内存管理优化

### 阶段七：测试与调试

1. 单元测试

   * Excel解析功能测试

3. 用户界面测试

   * 拖拽功能测试

   * 界面响应测试

### 阶段八：打包与部署

1. 使用PyInstaller打包
